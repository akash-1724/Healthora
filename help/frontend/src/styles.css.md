# File: `frontend/src/styles.css`

Global style sheet for the RBAC dashboard UI.

Style design goals:

- Implements a modern Neo-brutalism aesthetic.
- Bold borders, offset shadows (e.g., `4px 4px 0 #111827`), flat saturated colors.
- Chunky typography using the 'Outfit' font.
- Card-based dashboard blocks matching the brutalist theme.
- Responsive behavior on smaller screens with grid adjustments.

This file provides all the visual styling for the landing page, login, and the main application dashboard, enforcing a consistent and engaging UI without relying on external CSS frameworks.
