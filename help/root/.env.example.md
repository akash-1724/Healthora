# File: `.env.example`

Template environment variables.

- `POSTGRES_DB`, `POSTGRES_USER`, `POSTGRES_PASSWORD`: DB credentials.
- `POSTGRES_HOST_PORT`: host machine port mapped to container's `5432` (default `5433`).
- `DATABASE_URL`: SQLAlchemy DB connection string used by backend.
- `VITE_API_BASE_URL`: frontend API base URL (`http://localhost:8000`).
- `JWT_SECRET_KEY`, `JWT_ALGORITHM`, `JWT_EXPIRE_MINUTES`: JWT settings for auth.

Copy this file to `.env` before running docker compose.
